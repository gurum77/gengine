Detailed Description

  • 간단한 수학 계산관련
  • 입력값에 대한 예외처리는 하지 않는다.
    • 속도위주의 함수들이기 때문
    • caller가 예외처리를 해야함(여기서 예외처리 함수를 제공할 수는 있다)
