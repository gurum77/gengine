Detailed Description

  • import, export시 interface 역할을 한다.
  • 소스파일에 interface를 정의한 이유 - 다른 project가 interface를 참조할 필요가 없음.(header에 정의해봐야 참조도 안하는데, 빌드타임만 잡아 먹을 것 같아서 소스에 정의함)
