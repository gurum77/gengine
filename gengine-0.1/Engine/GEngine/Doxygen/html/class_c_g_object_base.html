Detailed Description

  • object의 기본 클래스
  • Object 개발시 주의 사항 -------------------------—
    • GDEFINE_REGISTE_OBJECT 를 헤더에 정의해서 object를 등록한다.
    • GIMPLEMENT_POOL, GIMPLEMENT_OPERATOR_COPY_BEGIN, GIMPLEMENT_OPERATOR_COPY_END 를 소스에 정의한다.(GDEFINE_REGISTE_OBJECT와 항상 커플로 필요하다)
