Detailed Description

  • action base 클래스
  • user defined action을 만들기 위해서는 반드시 상속받아야 한다.
  • action 등록 방법
    • action 개발시 : action header에서 GDEFINE_REGISTE_ACTION 사용해서 등록
    • action 사용시 : 사용할 프로젝트에서 GIMPLEMENT_REGISTE_ACTION 사용해서 등록
