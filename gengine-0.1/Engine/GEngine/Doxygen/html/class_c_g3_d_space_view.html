Detailed Description

  • space view
  • CG3DSpaceView는 CG3DView에 비해 대용량처리가 가능하지만 속도가 느리고, 멀티뷰가 지원되지 않는다는 단점이 있다.
  • 사용법
    • CG3DView 와 동일한 과정을 거침
    • CGSelectManager 설정
      • space 선택을 지원하기 위해서는 select manager가 설정되어야 한다.
